# 💫 About Me:

Software Engineer at **Esyasoft Technologies**, currently building a production-grade conversational AI platform for energy utilities — powered by **GPT-O 120B** served via **vLLM**, with an end-to-end multilingual voice pipeline (Whisper STT → LLM → Coqui TTS) achieving sub-4s round-trip latency across English and Indic languages.

Previously contributed to **Odoo 18** as part of the upstream R&D team, shipping features across Payroll, Recruitment, Time Off, and Expenses — code that reaches tens of thousands of customers globally. Before that, spent 1+ years at **TCS** administering enterprise Linux infrastructure for Qantas airline operations.

MCA graduate from **NIT Jamshedpur**. Passionate about systems that scale, AI that ships, and interfaces that feel effortless.
